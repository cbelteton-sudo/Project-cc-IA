# Project Structure Map

This document outlines the high-level structure of the `apps/` directory.

## `apps/api`

- **Purpose**: Backend API Server
- **Stack**: NestJS (Node.js)
- **Description**: Handles business logic, database interactions via Prisma, and API endpoints.

## `apps/web`

- **Purpose**: Frontend Web Application
- **Stack**: React, Vite, TypeScript
- **Description**: The user interface for the Construction Management system.
